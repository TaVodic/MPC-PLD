Text úlohy

Následující VHDL kód popisuje část stavového automatu z diagramu. Vyberte, které části diagramu kód popisuje.

process (clk) is
begin
  if rising_edge(clk) then
    case pres_state is
      when st_idle =>
        if start_i = '1' then
          pres_state <= st_data;
        end if;
      when st_data =>
        if enable = '1' then
          pres_state <= st_stop;
        end if;
      when st_stop =>
        pres_state <= idle;
      when others => null;
    end case;
  end if;
end process;

Úloha 3 Vyberte jednu nebo více možností:
a.

při nedodržení setup a hold time může nastat metastabilita na výstupu klopného obvodu

b.

hold time je čas po aktivní hraně hodinového signálu, po který nesmí dojít ke změně vstupu klopného obvodu

c.

f.

Reakce

Vaše odpověď je částečně správná.

Vybrali jste správně 1.
Správné odpovědi jsou: hold time je čas po aktivní hraně hodinového signálu, po který nesmí dojít ke změně vstupu klopného obvodu, při nedodržení setup a hold time může nastat metastabilita na výstupu klopného obvodu
